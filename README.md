# 🗂️ Portafolio Profesional — Paul André Negrete

Sitio web de portafolio y cotizador de servicios profesionales en **Datos, Logística & Tecnología**.

🌐 **[Ver sitio en vivo](https://pnegrete0209-web.github.io/paul-negrete-portfolio)**

---

## ✨ ¿Qué incluye?

- **Hero section** con estadísticas clave y llamados a la acción
- **6 tarjetas de servicios** con tecnologías y precios desde
- **Sección de proyectos** con SERPENS v5.0 y ÁGORA LMS
- **Cotizador inteligente** — el cliente selecciona servicios, complejidad, modalidad y urgencia; el estimado se calcula internamente y se envía por WhatsApp o correo con formato profesional
- **Sección Sobre mí** con certificaciones y habilidades
- Diseño 100% responsivo, oscuro y premium

---

## 🛠️ Stack

- HTML5 + CSS3 + JavaScript vanilla
- Fuentes: [Syne](https://fonts.google.com/specimen/Syne) + [DM Sans](https://fonts.google.com/specimen/DM+Sans) vía Google Fonts
- Sin frameworks, sin dependencias externas
- Un solo archivo `index.html` — listo para desplegar en cualquier host estático

## ✏️ ¿Cómo editar el contenido?

Abre `index.html` en GitHub con el lápiz ✏️ y usa `Ctrl+F` para encontrar:

| Qué cambiar | Buscar |
|---|---|
| Precios de servicios | `Desde $` |
| Descripción de servicios | texto dentro de `<p>` en cada `service-card` |
| Teléfono WhatsApp | `573008985321` |
| Correo electrónico | `infiniti.logconsultores@outlook.com` |
| Certificaciones | `cert-name` |
| Texto principal del hero | `Transformo procesos` |
| Proyectos | sección `id="proyectos"` |
